# Slumber Number

Slumber Number is a sleep timer app that lets users set a countdown and receive periodic audio notifications showing how much time has passed since the timer started or since the last alert.

## Features 

-Set a timer duration
*Set notification intervals
+Start, stop, and reset the timer
-Receive voice announcements of elapsed time

## Technologies
-React Native
*Expo
+react-native-timer-picker](https://www.npmjs.com/package/react-native-timer-picker)

## Installation

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

## Usage
Slumber Number was designed for people who want to track time while falling asleep. Users can set a timer duration and choose intervals for audio notifications that indicate how much time has elapsed since the start or since the last update.
While primarily intended as a sleep aid, the app can also be used for other timing needs such as workout tracking or study sessions.

## Project Structure
*components/ – Reusable UI components including custom buttons and picker elements, along with some Expo-provided components
*screens/ – Main application screens, including the home screen shown on launch
*styles/ – Shared styling used throughout the app
*Other files and folders are generated by Expo and support configuration and project setup
